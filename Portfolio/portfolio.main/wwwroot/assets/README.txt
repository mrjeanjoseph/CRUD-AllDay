Place your PDF resume and certification files here so they can be served by Blazor WebAssembly.

Expected default file names (you can change them in Layout/MainLayout.razor):
- resume.pdf
- certs/az-900.pdf, certs/az-204.pdf, certs/dp-900.pdf

Optional thumbnails (JPEG/PNG):
- certs/az-900-thumb.jpg, certs/az-204-thumb.jpg, certs/dp-900-thumb.jpg

Access URLs at runtime:
- /assets/resume.pdf
- /assets/certs/<file>.pdf
- /assets/certs/<file>-thumb.jpg
